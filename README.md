# 🚀 NeuroReel AI

AI-based YouTube Shorts controller using:

- 👁️ Eye Blink Detection → Pause/Play
- 👤 Head Movement → Scroll Control
- 🧠 Emotion Detection → Mood Analysis
- 📊 Dashboard → Analytics

## 🛠 Tech Stack
- Python
- OpenCV
- MediaPipe
- Transformers (HuggingFace)
- Tesseract OCR

## 🎯 Features
- Hands-free Shorts control
- Real-time emotion tracking
- Genre detection via OCR
- Interactive analytics dashboard

## ▶️ Run

```bash
pip install -r requirements.txt
python app.py
